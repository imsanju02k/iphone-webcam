🚀 Quick Start
1

Scan QR → Open in Safari

Use Safari on iPhone (not Chrome). GitHub Pages = HTTPS, so camera access works instantly — no warnings!

2

Allow Camera → Tap Start Streaming

Tap Allow when Safari asks. Choose 16:9 or 9:16 ratio, then tap Start Streaming.

3

Add OBS URL as Browser Source

In OBS: Sources → + → Browser → paste the OBS URL → set 1920×1080.

4

🎉 Live in OBS!

iPhone stream appears in OBS instantly. Works on any network, not just WiFi!
