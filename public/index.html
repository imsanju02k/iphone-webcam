🚀 Quick Start Guide
1

Scan QR → Open in Safari

Must use Safari. You'll see a “Not Secure” warning — tap Advanced → Visit Website (it's your own computer, safe to proceed).

2

Allow Camera → Start Streaming

Tap Allow when Safari asks. Then tap Start Camera → Start Streaming. App targets 4K UHD automatically.

3

Add Viewer URL to OBS

OBS → Sources → + → Browser Source. Paste the Viewer URL. Set 3840×2160 for 4K.

4

🎉 Streaming in 4K!

Your iPhone camera appears in OBS at up to 4K resolution. No extra apps needed!
